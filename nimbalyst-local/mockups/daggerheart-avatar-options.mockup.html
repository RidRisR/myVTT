护甲环设计规则

● 护甲环位于 HP/Stress 弧线与头像之间，像一圈实体甲片包裹角色
● 每片甲的弧度 = 360° / 总槽位数 - 间隙，槽位越多甲片越细密
● 完好甲片：金属蓝渐变 + 外侧高光边 + 微光投影
● 破损甲片：近乎透明，仅保留微弱轮廓（不是完全消失，暗示"这里曾有护甲"）
● 3槽 = 厚重板甲感，6槽 = 链甲/鳞甲，12槽 = 细密锁子甲
● 实际实现中通过 CSS 变量 --armor-total / --armor-current 动态计算

同心圆层次（由外到内）

① 符文环（最外，缓慢旋转，装饰性）+ Hope 宝石镶嵌其上
② HP 弧线（左半圈红色）/ Stress 弧线（右半圈紫色）— 从底部向上
③ 护甲环（蓝色甲片段）— 包裹在头像外
④ 头像（中心，金色边框 = 当前用户标识）
