## 1. 有格/无格模式

**决策**: 战术地图同时支持有格和无格两种模式，通过两个独立开关控制。

**两个独立属性**:

| 属性          | 控制                 | 默认值 |
| ------------- | -------------------- | ------ |
| `gridVisible` | 网格线是否可见       | true   |
| `gridSnap`    | Token 是否吸附到格子 | true   |

**组合效果**:

| gridVisible | gridSnap | 效果                       | 适用场景                |
| ----------- | -------- | -------------------------- | ----------------------- |
| ✅          | ✅       | 有格有吸附（经典战棋模式） | D&D、PF 等格子战斗      |
| ❌          | ❌       | 无格自由放置               | CoC、叙事战斗、自由摆放 |
| ✅          | ❌       | 有参考线但自由放置         | GM 参考距离但不强制对齐 |
| ❌          | ✅       | 隐形吸附                   | 不常用，但技术上支持    |

**对测量的影响**:

- 有格模式：测量以"格数"为单位显示
- 无格模式：测量以像素距离显示，骨架提供 `gridScale`（每格代表多少距离单位）配置，规则层负责换算为游戏内单位（英尺/米等）

## 2. 网格配置工具

**决策**: 分阶段实现。

**第一阶段（MVP）— 数值输入**:

- 战术工具条中弹出网格配置面板
- 配置项：gridSize（格子像素大小）、gridOffsetX/Y（偏移）、gridColor（颜色）
- 数值输入框 + 实时预览（修改即见效果）

**第二阶段（增强）— 可视化校准**:

- "校准网格"按钮进入校准模式
- 在地图上拖出两个点标记 "这段距离 = N 格" → 自动计算 gridSize
- 拖动网格整体调整偏移量
- 确认后退出校准模式

数值输入作为基础功能始终保留。

## 3. 测量工具

**决策**: 提供测距线和三种范围模板。

**工具列表**:

| 工具            | 交互                      | 显示                                            |
| --------------- | ------------------------- | ----------------------------------------------- |
| **测距线**      | 按住拖拽画线              | 线段 + 距离标注（有格: "6 格"，无格: 像素距离） |
| **范围 — 圆形** | 点击中心 + 拖拽半径       | 半透明圆 + 半径标注                             |
| **范围 — 锥形** | 点击起点 + 拖拽方向和长度 | 扇形（默认 90°）                                |
| **范围 — 矩形** | 拖出矩形                  | 半透明矩形                                      |

**持久性**:

- 默认：松开鼠标后测量结果立即消失（快速查看）
- Shift+拖拽：测量结果保留在地图上，通过 Awareness 广播给所有玩家（不持久化到 Yjs），手动清除

**骨架不做**:

- 距离单位换算（格 → 英尺/米）— 规则层接口
- 移动路径计算（考虑障碍物绕行）— 过于复杂

## 4. 光标同步

**决策**: 通过 Awareness 在战术地图上显示其他用户的鼠标位置。

**实现**:

- 通过 Awareness 广播鼠标位置（地图坐标系，非屏幕坐标）
- 显示：小箭头 + 用户名标签，颜色对应座位颜色
- 广播频率：~15fps（每 66ms 一次）
- 鼠标不在战术地图区域时不广播

**性能**: react-konva 迁移后，光标是 Canvas 图形，无 DOM 开销。

## 5. 迷雾/视野 ⏸

**决策**: 整体推迟到后续里程碑。

**推迟原因**:

- **动态视野**（根据 Token 位置自动计算可见区域）：需要墙壁编辑器、光线投射算法，是独立的里程碑级功能
- **GM 手绘迷雾**（手动遮盖/揭开区域）：即使是简化版也需要画笔/橡皮擦工具和图层管理，开发量不小

**当前替代**: GM 通过"玩家视角"按钮（议题 13）确认玩家能看到什么，配合 Token 可见性控制（GM-only Token）管理信息。

## 6. 战术工具条 UI

**决策**: 竖向工具条位于战术地图面板边缘，与面板一体（收起地图时一起消失）。

**工具条内容**:

| 按钮     | 功能                     | 备注                  |
| -------- | ------------------------ | --------------------- |
| 选择工具 | 点击选中/拖拽移动 Token  | 默认模式              |
| 测距工具 | 拖拽画测距线             |                       |
| 范围模板 | 子菜单：圆/锥/矩形       |                       |
| 网格开关 | 显示/隐藏网格 + 吸附开关 |                       |
| 网格配置 | 弹出配置面板             | gridSize/offset/color |
| 玩家视角 | GM ↔ 玩家视角切换        | 议题 13 决策          |
| 下一回合 | 推进先攻顺序             | 议题 13 决策          |
| 收起地图 | 关闭战术地图             | 回到叙事模式          |

**工具模式互斥**: 选择工具、测距工具、范围模板三者同一时间只能激活一个，当前激活的工具高亮显示。

## 7. 缩放控件

**决策**: 在战术工具条或地图角落补充缩放辅助控件。

| 控件           | 行为                     |
| -------------- | ------------------------ |
| **+ / − 按钮** | 点击按固定步长缩放       |
| **适应窗口**   | 一键将地图缩放至完整可见 |
| **回到中心**   | 一键将视口重置到地图中心 |

**位置**: 放在战术工具条底部或地图角落，具体位置在实现时调整。

## 8. 网格吸附预览

**决策**: Token 拖动过程中显示吸附预览。

**行为**: 拖动 Token 时，在最终将要吸附的网格位置显示一个半透明 ghost（Token 虚影），用户松开前即可看到 Token 将落到哪个格子。

**无格模式下**: 不显示 ghost（因为没有吸附，Token 落在松开位置）。
