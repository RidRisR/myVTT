## 1. 整体风格方向 — 深色 + 暖色调

**决策**: 对标 Alchemy RPG 的视觉方向。深色半透明面板 + 暖色调高亮，营造"烛光下的羊皮纸"感。

**核心约束**:

- 背景永远是氛围图 → 面板必须深色半透明
- UI 不能抢氛围图的视觉主角地位 → 克制、低调
- 所有背景色上的文字都必须有足够对比度 → WCAG AA 校验

**与当前方案的差异**:

- 当前：冷色调深蓝黑 `rgba(15,15,25,0.92)` + 蓝色高亮
- 新方向：暖棕黑 + 琥珀金高亮，整体更有奇幻氛围感

**后续扩展**: 未来可考虑提供多种 UI 主题风格（如冷色科幻、极简暗黑等），通过 CSS 变量切换。但当前只做暖色奇幻一套。

## 2. 粒子特效层

**决策**: 在氛围图之上、UI 面板之下增加一个粒子特效层，增强场景沉浸感。

**实现方式**:

- 独立全屏 `<canvas>`，`pointer-events: none`
- 轻量自写粒子系统（30-100 个粒子，不需要重型库）

**基础预设**:

| 预设     | 视觉效果                 |
| -------- | ------------------------ |
| 火焰余烬 | 橙红色微光粒子缓慢上升   |
| 飘雪     | 白色小点缓慢飘落         |
| 灰尘光斑 | 金色微粒随机漂浮         |
| 雨滴     | 斜线条快速下落           |
| 萤火虫   | 黄绿色小光点缓慢闪烁移动 |
| 无       | 默认，不显示粒子         |

**场景绑定**: 每个场景属性中增加 `particlePreset` 字段，GM 为每个场景选择合适的粒子效果。

**无障碍**: `prefers-reduced-motion` 下自动禁用。战术地图打开时可减弱或关闭。

## 3. 配色方案

**方向**: 全局偏暖，避免纯白和冷蓝灰。具体色值在实现时做 WCAG 对比度校验后微调。

| 语义 Token               | 色值参考                 | 用途                   |
| ------------------------ | ------------------------ | ---------------------- |
| `--color-bg-panel`       | `rgba(20,15,12,0.88)`    | 面板/卡片背景          |
| `--color-bg-surface`     | `rgba(35,28,22,0.92)`    | 输入框、嵌套区域       |
| `--color-border`         | `rgba(180,160,130,0.15)` | 面板边框、分隔线       |
| `--color-text-primary`   | `#F0E6D8`                | 主文字（暖白）         |
| `--color-text-secondary` | `#A09080`                | 辅助文字（暖灰）       |
| `--color-accent`         | `#D4A055`                | 高亮、选中态（琥珀金） |
| `--color-accent-hover`   | `#E8B86A`                | 高亮 hover（亮金）     |
| `--color-danger`         | `#C04040`                | 删除、伤害             |
| `--color-success`        | `#408040`                | 治疗、成功             |
| `--color-info`           | `#5090C0`                | 信息提示               |

## 4. 字体

**决策**: 当前阶段维持系统 sans-serif，用字重和大小区分层级。

**理由**: 额外字体增加加载时间，UI 文字可读性优先于个性。后续如需品牌感，可引入一个 Google Fonts 衬线体（如 Cinzel）用于 Logo 和大标题。

## 5. 面板视觉语言统一

**决策**: 所有面板/卡片遵循统一规范。

| 属性 | 值                                                      |
| ---- | ------------------------------------------------------- |
| 背景 | `var(--color-bg-panel)` + `backdrop-filter: blur(12px)` |
| 圆角 | `8px`                                                   |
| 边框 | `1px solid var(--color-border)`                         |
| 阴影 | `0 4px 24px rgba(0,0,0,0.5)`                            |

当前各组件各写各的值（有 6px、8px、12px 圆角混用），迁移时统一。

## 6. 图标规范

**决策**: 继续使用 Lucide，统一参数。

| 参数        | 值                                            |
| ----------- | --------------------------------------------- |
| strokeWidth | 1.5（统一，当前有 1/1.5/2 混用）              |
| 尺寸        | 16px（内联）、20px（UI 按钮）、24px（工具栏） |
| 颜色        | `currentColor`（跟随文字颜色）                |
